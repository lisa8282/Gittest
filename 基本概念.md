# 註解

* html 註解方法:
```javascript
  
<!-- 在這裡填寫註解內容 -->

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
```

* javascript 註解方法
``` javascript
 // 註解內容 主要用於一個單字的註解
 const myArray = []; // 這是一個陣列

-----------------------------

/**
*註解內容 主要用於 一個大範圍的註解
* 
*/
/**
* 按鈕按下後會做的事情
*
*/
funtion btn(){
  // 程式碼片段
}
-----------------------------

```

**說明**: 主要用來解釋這段程式寫什麼 無作用!

# 宣告
 * const 不可以跟改的宣告
 * let 可以更改的宣告
 * var 可以更改的宣告(此宣告因它會影響到程式碼，所以請習慣別用他!)

 範例: 
 ``` javascript
  const myBtn = ""; // 創造一個myBtn的變數
 ```
