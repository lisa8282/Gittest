## 在桌面創一個html 之後開始寫程式

``` javascript
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <!--網頁內容-->
  </body>
</html>

<script>
 // 程式碼內容
</script>
```

範例一:
輸入 一個數字 產生出 2、4、6、8....

``` javascript
  <!DOCTYPE html>
<html>

    <title>Page Title</title>
  </head>

  </body>
</html>

<script>
 // 程式碼內容
```
